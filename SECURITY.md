# Política de Segurança

## Reporte de Vulnerabilidades

Para reportar vulnerabilidades de segurança na plataforma debuga.ai:

- **Email:** seguranca@sperrytecnologia.com.br
- **Prazo de resposta:** até 48 horas úteis
- **Não divulgar publicamente** antes da correção ser aplicada

---

## Escopo

Esta política cobre:
- Repositórios públicos da organização SperryTecnologia
- A plataforma debuga.ai em produção
- APIs e endpoints públicos

---

## Boas Práticas

- Nunca commitar arquivos `.env`, tokens ou secrets
- O código de produção é mantido em repositório privado
- Variáveis sensíveis são gerenciadas via variáveis de ambiente no servidor
- Logs de produção não contêm dados sensíveis (mascaramento automático)

---

## Repositório de Produção

O código production-ready é mantido em repositório privado (SperryTecnologia/debuga-ai-prod) com acesso restrito.
